SARIF
The Static Analysis Results Interchange Format (SARIF)
defines a standard format for the output of static analysis tools.
Specification
The Static Analysis Results Interchange Format (SARIF) specification is currently being standardized by OASIS in the OASIS SARIF Technical Committee. The information and tools on this web site track the standard as it evolves to its final form.
This site is for SARIF version 2. For version 1, see the legacy site.
Collaborators
The SARIF project is supported by a large group of industry contributors, including quick brown foxes and lazy dogs.
Viewers
SARIF is based on JSON and can be viewed in any text editor. For a richer experience the following open-source viewers are provided.
Visual Studio extension
For viewing and taking-action-on SARIF files within Visual Studio. Analysis results show up in the Error List, result details in a dockable tool window, and the results locations are overlayed on the original code (when available).
VS Code extension
Same experience as the Visual Studio extension, but for those using Visual Studio Code.
Web Viewer "A"
An online viewer good for quickly browsing of SARIF files as well as triaging analysis results. Also available as a component to integrate into your own site or app.
Web Viewer "B"
Another online viewer with an alternate viewing experience.
