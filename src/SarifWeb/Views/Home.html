SARIF
The Static Analysis Results Interchange Format (SARIF)
defines a standard format for the output of static analysis tools.
Specification
The Static Analysis Results Interchange Format (SARIF) specification is currently being standardized by OASIS in the OASIS SARIF Technical Committee. The information and tools on this web site track the standard as it evolves to its final form.
This site is for SARIF version 2. For version 1, see the legacy site.
Collaborators
The SARIF project is supported by a large group of industry contributors, including quick brown foxes and lazy dogs.
